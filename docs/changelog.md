# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of HostKit
- Interactive TUI with Bubble Tea
- 22+ package support across 7 categories
- System Monitor with real-time stats
- Service Manager for start/stop/restart
- Log Viewer for system and service logs
- Backup Manager for configuration backups
- Firewall Manager (CSF/Fail2ban/UFW)
- Dependency resolution system
- Installation history tracking
- Auto-version detection from GitHub
- YAML-driven package configuration
- Multi-OS support (Ubuntu, Debian, AlmaLinux)

### Categories
- 🎛️ 7 Hosting Panels (aaPanel, CyberPanel, cPanel, Plesk, Hestia, CloudPanel, CWP)
- 🗄️ 5 Databases (MariaDB, MySQL, PostgreSQL, MongoDB, Redis)
- 🌐 3 Web Servers (Nginx, OpenLiteSpeed, PHP)
- 🔒 3 Security Tools (Fail2ban, CSF, Certbot)
- ⚙️ 2 Services (Docker, Portainer)
- 📊 1 Monitoring Tool (Netdata)
- 🔐 1 VPN Solution (WireGuard)

### Features
- One-click package installation
- System requirements validation
- Real-time installation progress
- Color-coded status indicators
- Package dependency resolution
- Configuration backup/restore
- Service status monitoring
- GitHub Pages documentation
- Automatic changelog generation
