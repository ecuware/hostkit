## Contributing

We love contributions! Whether you're fixing a typo, adding a new package, or improving the core functionality, please read our [Contributing Guide](CONTRIBUTING.md) to get started.

### Quick Start for Contributors

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/hostkit.git`
3. Read the [Development Setup](development/DEVELOPMENT.md) guide
4. Make your changes
5. Submit a pull request
