# HostKit

Welcome to **HostKit** - Your all-in-one hosting server management toolkit!

## 🚀 Quick Start

```bash
# Install
wget https://github.com/ecuware/hostkit/releases/latest/download/hostkit-linux-amd64
chmod +x hostkit-linux-amd64
sudo mv hostkit-linux-amd64 /usr/local/bin/hostkit

# Launch TUI
hostkit tui
```

## 🎛️ What is HostKit?

HostKit is a comprehensive toolkit for managing hosting servers, written in Go. It provides:

- **Interactive TUI** - Beautiful terminal interface
- **22+ Packages** - Panels, databases, web servers, security tools
- **One-Click Install** - Automated installation with dependencies
- **System Monitor** - Real-time CPU, RAM, Disk monitoring
- **Service Manager** - Start/stop/restart services
- **Backup & Restore** - Configuration backups
- **Firewall Manager** - IP blocking and security rules

## 📦 Supported Packages

### Panels
- aaPanel, CyberPanel, cPanel/WHM, Plesk, HestiaCP, CloudPanel, CWP

### Databases
- MariaDB, MySQL, PostgreSQL, MongoDB, Redis

### Web Servers
- Nginx, OpenLiteSpeed, PHP

### Security
- Fail2ban, CSF, Certbot

### Services
- Docker, Portainer

### Monitoring & VPN
- Netdata, WireGuard

## 💻 System Requirements

- **Ubuntu** 20.04+ (Recommended)
- **Debian** 11+
- 1GB RAM minimum (2GB+ recommended)
- Root access
